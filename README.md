# DataAnalytics_Projects
************************Predicting Student Health Risk*****************************

Projeto que faz a previsão do risco de saúde dos estudantes. Os dados de análise serão: health condition,	sleep duration,	heart rate,	bmi,	calorie expenditure,	step count,	exercise duration, water intake,	diet type,	stress level,	sleep quality,	physical activity level, smoking-alcohol and gender.
